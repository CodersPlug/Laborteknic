# Laborteknic — Presentación institucional

Slideshow HTML de una sola página con la presentación institucional de Laborteknic y sus líneas de negocio (BioSystems, Snibe, Techlab, Abbott Rapid Diagnostics — Toxicología), pensado para publicarse como sitio estático en GitHub Pages.

Contenido y fotos originados desde la carpeta de Google Drive "Presentaciones Institucionales Laborteknic".

## Estructura

- `index.html` — sitio completo (HTML + CSS + JS embebidos, sin build step).
- `assets/img/` — imágenes y logos optimizados para web (redimensionados/comprimidos).

## Navegación

- Flechas `←` `→` del teclado, barra espaciadora, botones en pantalla, swipe táctil o clic en los puntos inferiores.

## Publicar en GitHub Pages

1. Crear un repositorio en GitHub y subir este contenido.
2. En **Settings → Pages**, elegir la rama `main` y la carpeta raíz (`/`).
3. El sitio queda disponible en `https://<usuario>.github.io/<repo>/`.
